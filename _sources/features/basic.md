# HTML Roles

## Subscript & Superscript

```{raw} html
<myst-demo>
H{sub}`2`O
</myst-demo>
```

```{raw} html
<myst-demo>
4{sup}`th` of July
</myst-demo>
```

```{note}
These two roles are also accessible through `subscript` and `superscript`.
```
